# La HamburgueZona Website

Next.js website for La HamburgueZona.

## Quick edits

- Main page content: `app/page.tsx`
- Styling: `app/globals.css`
- Images: `public/images/`
- Order button URL: edit `ORDER_URL` at the top of `app/page.tsx`

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Upload this project to GitHub, then import the repo in Vercel.
